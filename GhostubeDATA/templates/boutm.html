Proposito de Ghostube

Este proyecto busca brindarte una experiencia muy comoda para ver tus propios videos con tu propio servidor ¡Totalmente local y aislado del internet!.
Hecho especificamente para esas personas que buscan una alternativa decente para la visualizacion de su propio contenido a traves de una interfaz visual intuitiva, amigable e interactiva.
¡Tambien hay control parental!

Uso de FFmpeg y FFprobe

Ambos se usan para una mejor previsualizacion de contenido (miniaturas y duracion de los videos). Pero son opcionales, Ghostube puede funcionar sin estos archivos.

Si descargaste Ghostube desde fuentes oficiales como github.com/PixelPiNa/Ghostube , entonces ya tienes incluidos ffmpeg.exe y ffprobe.exe en la carpeta "extenzzziones" dentro de GhostubeDATA, y no necesitas hacer nada mas. (Si estás usando linux, no necesitas esos archivos, los puedes borrar si quieres)

Instalacion

Te recuerdo que si lo descargaste desde fuentes oficiales, ya tienes incluidos los archivos necesarios y no necesitas instalar nada.

Para usuarios de Windows:

Puedes descargar ffmpeg y ffprobe desde el sitio oficial de ffmpeg: https://ffmpeg.org/download.html#build-windows,
o usar la descarga directa de abajo.

Descarga directa Github Latest Auto-Build (2026-06-09 15:17)

Después de descargar el archivo zip:
  1. Descomprime el archivo zip descargado.
  2. Entra a la carpeta descomprimida.
  3. entra a la carpeta "bin"
  4. Dentro de la carpeta "bin", encontrarás los archivos "ffmpeg.exe" y "ffprobe.exe".
  5. Copia solo los archivos "ffmpeg.exe" y "ffprobe.exe", ignora el otro.
  6. Pega los archivos copiados dentro de la carpeta "extenzzziones" que se encuentra dentro de "GhostubeDATA".

Ghostube\GhostubeDATA\extenzzziones


Para usuarios de Linux:

Ffmpeg y ffprobe suelen estar disponibles en los repositorios de tu distribución. Puedes instalarlos usando el gestor de paquetes de tu sistema. Por ejemplo, en Debian/Ubuntu puedes usar:

sudo apt-get install ffmpeg

En Fedora:

sudo dnf install ffmpeg
