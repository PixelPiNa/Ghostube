# Ghostube
Ghostube es un servidor web hecho en Flask que permite levantar tu propio sitio de video local basado en Tags

#Necesita FFMPEG Y PYTHON3 para funcionar
- Ffmpeg es OPCIONAL, la funcion de duracion del video lo requiere, pero si no tienes ffmpeg instalado, el servidor funcionará pero no mostrará la duración de los videos. Puedes descargar ffmpeg desde https://ffmpeg.org/download.html y agregarlo a tu PATH para que sea reconocido por el sistema.
- Python3 es totalmente necesario
