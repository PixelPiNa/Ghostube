🧹 Mantenimiento

Limpieza de Base de Datos:
Esto eliminará las etiquetas que no se usan en ningún video (tags huérfanos) y compactará el archivo de base de datos para ahorrar espacio.



🧨 Purgar Base de Datos (Rutas eliminadas):
Si borraste una carpeta recientemente o desconectaste un disco duro, sus videos siguen registrados en la base de datos de este servidor. Este botón eliminará los registros de esos videos para que no consuman espacio en la base de datos.
(Tus archivos no se borraran, solo los registros en este servidor).
